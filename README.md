# Netflix Clone

A Netflix-inspired streaming platform clone built using modern web technologies.

## Features
- User authentication
- Movie browsing UI
- Responsive design
- Modern frontend
- Fast performance

## Tech Stack
- React / Next.js
- JavaScript / TypeScript
- Tailwind CSS
- Node.js

## Installation

```bash
npm install
npm run dev
```

- components/
- pages/
- public/
- styles/
